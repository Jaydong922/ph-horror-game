# Sa Lilim ng Lumang Bahay
🇵🇭 Filipino Horror Game | 100% Mobile‑Built

A first‑person horror game inspired by Filipino folklore. Explore an old house, uncover secrets, and survive the unknown.

✅ Features:
- Dark atmosphere & fog effects
- Simple touch controls
- No ads, no tracking, completely free
- Made with HTML5 / Three.js
